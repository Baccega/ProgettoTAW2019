# Restaurant App (TAW Project 2019)

### Baccega Sandro, Scodeller Giovanni

Assignment for Web Technologies and Applications course.

## Dependencies

- Yarn
- Node.js 10.x
- Android Sdk (for the Android build)

## Building

```bash
yarn install

yarn start-backend

yarn start-web
yarn start-android
yarn start-desktop
```

Server run on http://localhost:4200
